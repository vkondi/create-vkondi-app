## [1.0.0] - 2026-02-12

### Added
- Initial release of create-scaffold-kit
- React (Vite) scaffolding with TypeScript support
- Next.js scaffolding with App Router
- ESLint configuration with flat config
- Prettier integration
- Husky + lint-staged for git hooks
- Vitest testing framework
- Tailwind CSS support
- Docker setup with multi-stage builds
- Docker Compose for full-stack applications
- GitHub Actions CI/CD workflows
- Automatic package manager detection
- Feature-based folder structure for React
- Path aliases configuration
- Strict TypeScript mode
- Modular, extensible architecture
- Comprehensive logging and error handling
- Git repository initialization
- Interactive CLI with prompts
- Production-ready configurations

### Features

#### Scaffolding
- React with Vite and clean boilerplate
- Next.js with security headers

#### Development Tools
- ESLint strict/standard modes
- Prettier with opinionated config
- Pre-commit hooks for linting and testing
- Pre-push hooks for type checking

#### Testing
- Vitest with React Testing Library
- Example tests
- Coverage reporting
- Interactive UI mode

#### Styling
- Tailwind CSS with PostCSS
- Configured for both React and Next.js

#### DevOps
- Multi-stage Docker builds
- Docker Compose orchestration
- GitHub Actions with multi-version testing
- Full CI/CD pipeline
